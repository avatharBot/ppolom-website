P'polom

This project aims to propose a model for an evolutive approach to get the best set of conditions to import a product to México. Given the number of variables that must be taken into account, the analysis easily grows in complexity, so an evolutive algorithm offers a reduction on complexity and a model that’s easy to modify to optimize performance, or even to change the way those optimal conditions are met
To test P'polom, select a product and send to see the proposed solution
